## 下载

### Docker

```
docker pull apachecn0/freelearn-db-zh
docker run -tid -p <port>:80 apachecn0/freelearn-db-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g freelearn-db-zh
freelearn-db-zh <port>
# 访问 http://localhost:{port} 查看文档
```
